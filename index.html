Interactive Charts

This is fun practice for C3.js. It's my first time learning how to create charts using C3.js and D3.js. I've seen some pretty cool examples.

C3 makes it easy to generate D3-based charts by wrapping the code required to construct the entire chart. We don't need to write D3 code any more. C3 gives some classes to each element when generating, so you can define a custom style by the class and it's possible to extend the structure directly by D3.C3 provides a variety of APIs and callbacks to access the state of the chart. By using them, you can update the chart even after it's rendered.
